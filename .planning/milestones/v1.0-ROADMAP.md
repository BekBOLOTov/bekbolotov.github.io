## Overview

Three-phase milestone took the project from empty repo to live, polished CV landing page on GitHub Pages. Phase 1 stood up the deployed shell so subsequent changes were verifiable on real URL. Phase 2 filled all content sections — hero through contact — delivering core credibility value. Phase 3 locked in mobile rendering, print-to-PDF output, SEO meta, and content refresh with inline tech stacks and 2-tier skills model.

Plain HTML/CSS/JS only, no build step, deploys directly from `main` to GitHub Pages.

## Milestone Summary

**Decimal Phases:** None — all integer phases, no urgent insertions.

**Key Decisions:**
- Plain HTML/CSS/JS over static generator — zero build step, deploys directly to GH Pages
- Single page (no routing) — recruiters scroll, not navigate
- Skip projects section in v1 — focus on credibility via experience
- 2-tier skills model (Expert + Proficient, no Familiar) — D-06/D-07 collapse during Phase 3
- Inline Stack lines per role — D-05, replaces separate stack column
- og-card.png path (lowercase, hyphenated) — D-22, overrode early STACK.md "og-image.png" mention
- No `print-color-adjust` in print CSS — D-15, lets browser strip backgrounds per default
- IntersectionObserver for active nav highlighting — Phase 2 plan 5
- Deploy directly from `main` branch root (`/`) — branching_strategy: none

**Issues Resolved:**
- Phase 2 UI-SPEC checker blockers (weights, tokens, spacing) — fixed in 03fc8b2
- Education section dropped (no formal degree) — replaced with achievements (5f79675)

**Issues Deferred (v2):**
- Projects section — keep v1 focused on credibility through experience
- Certifications under education — none yet
- Favicon — not blocking
- Custom domain (CNAME + DNS) — github.io URL sufficient for v1
- Dark mode toggle — adds complexity without clear need

**Technical Debt Incurred:**
- `.DS_Store` in working tree, not yet in `.gitignore`
- Hardcoded #000/#fff/#444 in @media print — intentional (print-context overrides), but not documented in style.css comments

**Performance Targets (Phase 3 verification):**
- LCP < 1.5s — achievable with no render-blocking JS, font-display: swap, no image weight in hero
- Total page weight < 200kB — index.html + style.css + script.js + og-card.png (102KB) well under target
