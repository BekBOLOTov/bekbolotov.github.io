## Outcome Notes

**Adjusted during milestone:**
- "Education" replaced with "Achievements" early (5f79675) — engineer has no formal degree; competitive programming honors carry the same credibility signal. EDU-01 remained mapped to Achievements section.

**Tier model evolved:**
- SKILL-01 originally listed "Expert / Proficient / Familiar". During Phase 3 D-06/D-07, collapsed to 2 tiers (Expert + Proficient) with all Familiar items merged into Proficient. Honest-tier intent preserved with cleaner visual hierarchy.

**Stack signal added (D-05):**
- EXP-03 (tech stack per role) initially planned as Skills section coverage. Phase 3 added inline `.experience-stack` line per `.experience-entry` for stronger per-role signal.

## Out of Scope (Permanent)

- Backend or CMS — static only, content hardcoded
- Contact form — links only, no server-side handling
- JavaScript animations, scroll effects, typed text — compete with readability
- Skills progress bars — visually misleading, undercut credibility
- Multiple pages or routing — single scrollable page only
