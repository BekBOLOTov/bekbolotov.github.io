  <action>
Append the following @media print block at the very end of style.css, after the closing } of @media (max-width: 600px). This must be the last block in the file.

@media print {
  /* Hide screen-only chrome */
  header,
  footer {
    display: none;
  }

  /* Strip section dividers (visual only, not structural) */
  section {
    border-top: none;
  }

  /* Force black text on white background */
  body {
    color: #000;
    background: #fff;
  }

  /* Links: readable but not blue — underline makes them still recognizable as links */
  a {
    color: inherit;
    text-decoration: underline;
  }

  /* Append full URL after every link */
  a[href]:after {
    content: " (" attr(href) ")";
    font-size: 0.85em;
    color: #444;
  }

  /* Suppress URL expansion for non-useful link types */
  a[href^="mailto:"]:after,
  a[href^="#"]:after {
    content: none;
  }

  /* Page-break control — both legacy and current property for cross-browser */
  .experience-entry,
  .achievements-group,
  .skills-tier {
    page-break-inside: avoid;
    break-inside: avoid;
  }

  /* Standard page margins */
  @page {
    margin: 1in;
  }
}

IMPORTANT: The hardcoded values #000, #fff, #444, 0.85em, and 1in are intentional print-specific overrides — they are not design-system values and must NOT be replaced with CSS custom properties from :root. The :root tokens are screen-context values and do not apply in print.

Do NOT add -webkit-print-color-adjust: exact or print-color-adjust: exact — this would preserve background colors in print, which D-15 explicitly prohibits.
