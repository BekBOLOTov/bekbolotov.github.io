## Spacing Scale

All spacing tokens already declared in `style.css` `:root`. Executor MUST use CSS variables — no hardcoded values.

| CSS Token | Value | Usage |
|-----------|-------|-------|
| `--space-1` | 4px | Inline gaps: tag margin, icon-to-text gap |
| `--space-2` | 8px | Tight element spacing: tag padding, muted label gap |
| `--space-3` | 12px | Nav link padding, skill tier label margin |
| `--space-4` | 16px | Default paragraph margin, list item spacing |
| `--space-6` | 24px | Experience entry gap, sub-section spacing |
| `--space-8` | 32px | Section internal padding (top/bottom) |
| `--space-12` | 48px | Between major sections |
| `--space-16` | 64px | Hero top/bottom padding |

Exceptions: Nav bar height target is 48px (touch-target safe); achieved via `padding: var(--space-3) var(--space-6)` on `<header>`.

## Typography

Tokens already declared in `style.css`. Map to roles below.

| Role | CSS Token | Size | Weight | Line Height | Usage |
|------|-----------|------|--------|-------------|-------|
| Display | `--font-size-2xl` | 32px | 700 | 1.2 | Name (`<h1>`) — single use |
| Heading | `--font-size-xl` | 24px | 600 | 1.3 | Section `<h2>` headings |
| Body | `--font-size-base` | 16px | 400 | `--line-height` (1.6) | Summary, bullets, body copy |
| Label / Muted | `--font-size-sm` | 14px | 400 | 1.5 | Dates, location, muted metadata, tag text |

Font family: always `var(--font-body)` — `'Inter', system-ui, -apple-system, sans-serif`.

No italic. No underline except on links (underline on hover only for nav; always underlined for inline content links).

## Color

All color tokens already declared in `style.css` `:root`.

| Role | CSS Token | Value | Usage |
|------|-----------|-------|-------|
| Dominant (60%) | `--color-bg` | #ffffff | Page background, all section backgrounds |
| Secondary (30%) | `--color-text` | #1a1a1a | All body text, headings, bullets, labels |
| Muted text | `--color-text-muted` | #6b7280 | Dates, location, job title subtitle in experience, tag text |
| Border / Divider | `--color-border` | #e5e7eb | Section dividers (`border-top`), skill tag borders |
| Accent (10%) | `--color-accent` | #2563eb | Reserved list below — nowhere else |

**Accent reserved for (exhaustive list — D-16):**
- External links: email `href`, GitHub `href`, LinkedIn `href`, DS&A profile links, IOI stats link
- Nav `<a>` links on hover/focus state only (not default state)
- Active nav link indicator (underline or `border-bottom`)

Accent is NOT used for: headings, section backgrounds, skill tags, badges, or decorative elements.

No destructive actions exist in Phase 2. No red/warning color needed.

## JavaScript Contract

One optional JS behavior: active nav link highlighting via `IntersectionObserver`.

File: `script.js` (create if absent, add `<script defer src="script.js"></script>` before `</body>`).

```
Behavior: observe #experience, #skills, #achievements, #contact.
When a section is >=20% visible, add class "active" to the matching nav <a>.
Remove "active" from all others.
No other JS in Phase 2.
```

If `script.js` is not created, the page is fully functional — nav hover state is sufficient. This is strictly progressive enhancement.
